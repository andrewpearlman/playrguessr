# MLB Playr Guessr

A browser-based baseball trivia game. A mystery player's career batting 
statistics are displayed (without their name), and you try to identify the player.

## How to play
Visit: https://andrewpearlman.github.io/playrguessr/MLB-playr-guessr.html

## Data
Data includes hitters only, for now.
Player data sourced from Fangraphs (standard and dashboard batting stats, 
fielding). Covers MLB hitter seasons from 1985–2025. Eligibility filter: 5+ seasons 
from 2000-2025, including 2+ seasons with 100+ games played in that span.
